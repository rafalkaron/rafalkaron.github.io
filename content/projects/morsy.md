## Technologies

I used the following technologies to develop Morsy:

* HTML
* CSS
* Vanilla JavaScript

Due to webkit limitations, Morsy does not fully support web browsers on iOS.

## Features

Morsy includes the following features:

* Instant conversion
* Playback controls
* Light signals
